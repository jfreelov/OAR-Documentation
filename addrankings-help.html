AddRankingsToTable

Sorts an Excel table using priority rules written above the header, then appends OverallRank (and optionally CategoryRank) columns.

Setup

  1. Write directives in the row directly above the table header. One signed integer per column you want to sort by, blank otherwise.
    |n|
    priority: 1 = primary, 2 = tiebreak, 3 = next…
    sign
    + ascending, − descending
  2. (Optional) click a header cell to group CategoryRank by that column. Clicking anywhere else inside the table → only OverallRank.
  3. Run the macro. Rank column(s) appear at the right edge.

Good to know

  • At least one numeric directive is required.
  • Ties fall through to the next priority, then the next…
  • Active cell must be inside the table when you run the macro.
  • CategoryRank is added only when the active cell is on a header cell.
  • New columns are named OverallRank and CategoryRank.

If it errors

  • Active cell is not inside a table. Click a cell inside the Excel table before running.
  • No ranking directives found in the row above the table. Add at least one signed integer directly above a header cell.
  • Table has no data rows. The table body is empty — add data first.
  • Active cell is not in a valid table header column. Active cell is on the header row but outside the table's column range.
